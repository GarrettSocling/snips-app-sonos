Setup Instructions

  • Create a configuration file config.ini at the root of the snips-app-sonos folder
  • Make sure the configuration file has the following fields, in the global section
    • hostname={{hostname}}

Accounts

Spotify

  • Head over to your Spotify Developper Dashboard
  • Create an application, give it a name, and add a Redirect URIs that has the following structure :
    • http://{{hostname}}:5000/callback/
  • You are now ready to edit your configuration file on your raspberry pi
  • Make sure the configuration file has the following fields, in the global section
    • redirect_uri
  • Make sure the configuration file has the following fields, in the secret section
    • client_id
    client_secret
    redirect_uri
